# ScrollBar

pc端滚动组件，自动适应父级容器宽度，超出部分进行滚动，注:需要一个父级容器并给于宽度，文本类型请设置不换行即可,style必填项目。

# API

属性 说明 默认值
noresize 如果 container 尺寸不会发生变化，最好设置它可以优化性能 Boolean
horizontalClass 水平滚动条容器className名 String
verticalClass 垂直滚动条容器className名 String
wrapClass 内联方式 自定义content容器的className
viewClass 内联方式 自定义view容器的className
viewStyle 内联方式 自定义view容器的样式
always 滚动条是否一直显示 false
callback 滚动回调 回调参数event
scrollTo 滚动至对应位置根据ID进行处理，目前仅支持垂直滚动 this.$refs.scrollBarName.scrollTo(id)
