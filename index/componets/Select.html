# select 选择器

用来替换仿原生的选择器,扩展了单选，多选禁用选择

# 基本用法

使用 v-model 双向绑定数据。

  • New York
  • London
  • Sydney

    <iSelect v-model="value">
      <Opction value="New York" label="New York">New York</Opction>
      <Opction value="London" label="London">London</Opction>
      <Opction value="Sydney" label="Sydney">Sydney</Opction>
    </iSelect>

# 尺寸

设置size为 small 或 large 使用大号和小号的开关,默认为中号

  • New York
  • London
  • Sydney
  • New York
  • London
  • Sydney
  • New York
  • London
  • Sydney

    <iSelect v-model="value" size="small">
          小号
    </iSelect>
    <iSelect v-model="value">
          中号
    </iSelect>
    
     <iSelect v-model="value" size="large">
          大号
    </iSelect>

# 自定义内容

  • New York America
  • London U.K.
  • Sydney Australian

    <iSelect v-model="value">
         <Opction value="New York" label="New York">
            <span>New York</span>
            <span style="float:right;color:red">America</span>
        </Opction>
        <Opction value="London" label="London">
            <span>London</span>
            <span style="float:right;color:#ccc">U.K.</span>
        </Opction>
        <Opction value="Sydney" label="Sydney">
            <span>Sydney</span>
            <span style="float:right;color:#ccc">Australian</span>
        </Opction>
    </iSelect>

# API

属性 说明 默认值
noresize 如果 container 尺寸不会发生变化，最好设置它可以优化性能 Boolean
horizontalClass 水平滚动条容器className名 String
verticalClass 垂直滚动条容器className名 String
wrapClass 内联方式 自定义content容器的className
viewClass 内联方式 自定义view容器的className
viewStyle 内联方式 自定义view容器的样式
always 滚动条是否一直显示 false
callback 滚动回调 回调参数event
scrollTo 滚动至对应位置根据ID进行处理，目前仅支持垂直滚动 this.$refs.scrollBarName.scrollTo(id)
