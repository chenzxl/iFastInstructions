# Icon

ifast的图标使用开源项目 ionicons1.5.2使用组件，指定图标对应的type属性，示例代码：

  <Icon type="ion-ionic"></Icon>
  <Icon type="ion-ionic" size="20" color="red"></Icon>
  <Icon type="ion-ionic" color="#cccc"></Icon>
属性 说明 类型 默认值
type 图标的名称 String 无
size 图标的大小，单位是px Number String 无
color 图标的颜色 String 无
