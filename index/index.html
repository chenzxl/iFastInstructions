# 启程

为什么写了这个iFast-UI组件库，起因是去年快过年的时候，我的好友（你猜?），闲聊的时候，闲的时候就别打游戏，玩什么王者了，干点有意义的事情，可以试着组队写个组件库，说干就起（也是闹着玩），于是乎，我又一次拾起对代码乐情，在网上看资料，翻看Bootstrap,iview,element,vue等各大框架源码，css方面，因为之前用的就是Bootstrap，样式方面就参照他了，第一次整这东西,代码许多次推倒重来，其中的艰辛就不说了，目前只完成10多个组件开发，各种原因，文档没有及时更新到，粗略算了下还有30多个需要开发的，从初创两人，中间有人加人，到现在本人自己开发。

路漫漫其修远兮，吾将上下而求索，不舍望会火起来，东西还很多，一个好的组件需要，认真思考和规划，在时间允许的情况下，希望在年中后期完成，就当一次历练！
